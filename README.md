# 🔍 Sistema de Identificação de Placas Veiculares com YOLOv8

Projeto desenvolvido para **detecção automática de placas de veículos** que acessam o campus universitário, utilizando técnicas de **Visão Computacional**, **Deep Learning** e **YOLOv8**.

---

- Visão Geral do Projeto  
- Arquitetura Geral  
- 1️⃣ Treinamento do Modelo  
- 2️⃣ Rastreamento (Tracking)  
- 3️⃣ Pré-Processamento das Imagens  
- 4️⃣ Leitura da Placa (OCR)  

---

## Visão Geral do Projeto

Este projeto implementa um sistema completo capaz de:

- ✔ Detectar veículos  
- ✔ Isolar e identificar placas veiculares  
- ✔ Realizar OCR para extrair o texto da placa  
- ✔ Operar em tempo real ou processar vídeos gravados  

O objetivo é **automatizar o controle de entrada no campus**, reduzindo intervenção humana e aumentando a segurança.
